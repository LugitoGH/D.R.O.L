O **GitHub** é uma plataforma de hospedagem e colaboração de código baseada em Git. Ele permite que equipes versionem, compartilhem e mantenham projetos de forma organizada, acompanhando alterações ao longo do tempo.

Entre suas principais ferramentas estão:

- **Repositórios**: armazenam o código-fonte e arquivos do projeto.
    
- **Controle de versão (Git)**: registra alterações, permitindo histórico, comparação e reversão de mudanças.
    
- **Branches**: possibilitam desenvolvimento paralelo sem afetar a versão principal.
    
- **Pull Requests**: permitem revisão de código antes de integrar mudanças.
    
- **Issues**: organizam tarefas, bugs e discussões.
    
- **Actions**: automatizam testes, builds e deploys.
    
- **Projects**: auxiliam no gerenciamento de tarefas em formato de quadro.
