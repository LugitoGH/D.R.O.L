## O que é Docker?

O Docker é uma plataforma aberta para desenvolver, distribuir e executar aplicações. O Docker permite separar suas aplicações da sua infraestrutura, possibilitando a entrega rápida de software. Com o Docker, você pode gerenciar sua infraestrutura da mesma forma que gerencia suas aplicações. Ao aproveitar as metodologias do Docker para distribuição, teste e implantação de código, você pode reduzir significativamente o tempo entre escrever o código e executá-lo em produção.
